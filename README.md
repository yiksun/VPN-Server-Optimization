# VPN-Server-Optimization
VPS性能优化脚本
  1.  自动检测并优化（推荐）
  2.  小内存优化方案（<=2GB）
  3.  中等配置优化方案（2-4GB）
  4.  大内存优化方案（>4GB）
  5.  仅设置BBR
  6.  查看当前网络配置
  7.  恢复默认配置
  8.  Swap管理
       1.  检查当前swap状态
       2.  创建/修改swap
       3.  关闭并删除swap
  0.  退出脚本


```
wget -O optimize.sh "https://github.com/zmethan/VPN-Server-Optimization/raw/main/optimize.sh" && chmod +x optimize.sh && ./optimize.sh
```
